Data Visualization in Python (Amazon's Books dataset)

As mentioned, I have utilized Plotly packages. However, there are some technical issues. As seen on my github where I post the code repository, the visualization using Plotly packages are not displayed. This is because github performs a static render of the notebooks and it does not include the embedded HTML/JavaScript that makes up a plotly graph.

I'm still working on this issue but, for now, please note that the graphs shown below are the Plotly graphs.

The table shows the top 10 authors with the highest average rating. In the repository code, this visualization is called by fig_1.

Code

fig_1 = make_subplots(rows=1, cols=1, subplot_titles=("")) 
# Setting table parameters
fig_1.add_trace(go.Table(columnorder = [1,2],
				columnwidth = [35,35],
				header=dict(values=top_10_authors.columns, 
				line_color='black',
				fill_color='mediumaquamarine',
				height=30),
				cells=dict(values=[top_10_authors['Author'],
				top_10_authors['User Rating']], 
				line_color='black',
				fill_color='White')))
										
# Setting the parameters of the chart when displaying
fig_1.update_layout(showlegend=False, 
					title_text='Top 10 best authors', 
					title_font_size=25, 
					title_font_family='Arial Rounded MT Bold', 
					title_x=0.5,
					font=dict(family='Arial', 
					size=12, 
					color='black'))
										
# Displaying the graph
fig_1.show()

The donut chart shows the percentage of the books genre (Non-Fiction and Fiction). In the repository code, this visualization is called by fig_2.

Code

# Let's set the visualization parameters
fig_2 = make_subplots(rows=1, cols=1)
										
# Setting table parameters
fig_2.add_trace(go.Pie(values=number_of_books_by_genre['Name'],
				labels=['Non Fiction', 'Fiction'],
				textinfo='label+percent',
				insidetextorientation='radial',
				hole=0.3,
				marker=dict(colors=['Gray','mediumaquamarine'],

				line_width=1),
				textfont={'color': 'azure', 'size': 17},

					showlegend=False, 
					title_text='Count of books genre', 
					title_x=0.5, 
					font=dict(family='Arial', 
					size=15, 
					color='black'))

fig_2.show()
